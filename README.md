Step 1: Login Screen
When you open the app, the first thing you'll see is a login screen. If you're new, don't worry! There's a "Sign Up" option.Click on it. Upon launching the application, you're greeted with a user-friendly login screen. This is the gateway to the app's features. Here, users need to input their previously registered credentials — a unique username and a password. If you're a new user, you can opt to register by selecting the "Sign Up" option.

Step 2: Sign Up
Here, you need to register your email, create a username, and set up a password. Once done, go back to the login screen. Clicking on "Sign Up" takes you to the registration page. Here, you'll need to provide your email address, a unique username that distinguishes you within the application, and a password for account security. This information is crucial for user verification and ensuring that each user has a distinct identity within the system.

Step 3: Login
Now, use your newly created username and password to log in. If everything checks out, you'll be taken to the main activity. Returning to the login screen with your newly created credentials, you input your username and password to gain access to the app. This authentication step ensures that only authorized users can utilize the app's features.

Step 4: Main Activity
Welcome to the main screen! Here, you'll see subjects you can give feedback on. Choose a subject that you want to provide feedback for. Upon successful login, you're ushered into the main activity of the application. This central hub serves as the starting point for your interaction. Here, you'll find a curated list of subjects or courses for which feedback can be provided. This design allows users to navigate easily to the area they want to provide feedback on.


Step 5: Question Page
Clicking on a subject takes you to a page with questions related to that subject. Read the questions carefully and give your answers. Clicking on a specific subject takes you to a dedicated page tailored to that subject. This page is designed to streamline the feedback process by focusing on one subject at a time. It ensures a more organized and user-friendly experience.

Step 6: Submit Feedback
After answering all the questions, hit the submit button. Your valuable feedback is now recorded!. On the subject-specific page, you'll encounter a series of carefully crafted questions related to the chosen subject. These questions are structured to gather comprehensive feedback, encouraging users to express their thoughts, concerns, and suggestions in a systematic manner. The goal is to capture detailed insights that can be valuable for improvement.

Step 7: Logout
Once you're done giving feedback, simply click on the logout button. It keeps your information safe. After responding to all the questions, the user reaches the final step — submitting feedback. The submit button serves as a conclusive action, signaling that the user has completed the feedback process. This action securely sends the feedback data to the system for storage and analysis..Once the feedback is submitted, it's essential to log out for security purposes. Logging out ensures that the user's account is protected, especially if the app is being used on a shared device. This step adds an extra layer of security to the user's account information.

Bonus: Admin Portal
On the main login screen, there's a special area for admin. Admins can log in here to see all the feedback given by students. It helps them keep track of what's happening. The main login screen also features a dedicated space for administrators. Admins can log in using their unique credentials to access a specialized portal. Within this portal, administrators gain a comprehensive overview of all the feedback provided by students. It acts as a control center, allowing admins to monitor and analyze feedback data efficiently.
